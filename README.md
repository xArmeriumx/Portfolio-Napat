# Napat Pamornsut - Portfolio

Personal portfolio website showcasing my projects, skills, and experience.

## Live Demo

[https://napat-dev.vercel.app](https://napat-dev.vercel.app)

## Features

- Responsive Design - Works on all devices
- Project Showcase - Detailed project pages with image galleries
- Skills Section - Organized by category (Frontend, Backend, Testing)
- SEO Optimized - Custom meta tags for each page
- Fast Performance - Built with Vite for optimal loading

## Tech Stack

- **React** - UI Library
- **Vite** - Build Tool
- **React Router** - Navigation
- **CSS** - Custom styling

## Project Structure

```
src/
├── components/     # Reusable components (Navbar, Card, Section)
├── pages/          # Page components (Home, About, ProjectList, ProjectDetail)
├── data/           # Static data (profile.js, projects.js)
├── hooks/          # Custom hooks (usePageMeta)
└── styles/         # CSS files
```

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

- **Vercel** - Auto-deploy from main branch
- **GitHub Pages** - Manual deploy with `npm run build`
